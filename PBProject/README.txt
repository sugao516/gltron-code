Quick ReadMe for GLTron Development with Project Builder


Debugging:

	Select the Development build style and recompile to debug. (Under the "Targets" vertical tab). This enables debugging symbols and disables all optimizations.

Running the compiled binary:
	
	If you want to run the binary from Project Builder, you have to set the working directory of the executable to GLTron.app/Contents/Resources (use cmd-option-x to edit the current executable)
